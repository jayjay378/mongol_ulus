# 사운드 (효과음 · 배경음악)

이 폴더에 오디오 파일을 넣으면 게임이 자동으로 인식해 재생합니다.
**코드 수정·재빌드 불필요** — 파일을 넣고 앱을 새로고침(배포는 커밋·푸시)하면 됩니다.

## 인식하는 파일 이름

아래 이름 중 하나로 저장하세요(확장자는 `.mp3` · `.ogg` · `.wav` · `.m4a` 중 아무거나, 같은 이름이면 이 순서로 우선).

| 파일 이름 | 언제 재생되나 |
|---|---|
| `page` | **책 넘기는 소리** — 연출 카드가 넘어갈 때(오프닝 독백·도시 풍경·보고 등 서사 카드 전환, 백과사전 항목 열기) |
| `bgm`  | **배경음악** — 첫 클릭 후 시작, 반복 재생(🔊 버튼으로 음소거) |
| `select` | (선택) 타이틀에서 모드(스토리/샌드박스)를 고를 때 |
| `arrive` | (선택) 역참로에서 도시에 도착할 때 |
| `star` | (선택) 채점·별점이 나올 때 |

예) `page.mp3`, `bgm.mp3`, `select.wav` …

## 주의

- 파일은 **base64로 페이지에 내장**되어 별도 서버 설정 없이 로컬·배포 모두 동일하게 동작합니다.
- 그래서 **너무 큰 파일은 피하세요**: 효과음은 보통 수십 KB, 배경음악은 **2MB 이하의 짧은 루프**를 권장합니다
  (스토리 모드는 한 번만 불러오지만, 샌드박스에서 사이드바를 바꾸면 다시 불러오므로 클수록 느려집니다).
- 브라우저 정책상 소리는 **사용자가 화면을 한 번 클릭(모드 선택 등)한 뒤** 재생됩니다.
- 우상단 **🔊/🔇 버튼**으로 전체 음소거를 토글합니다(설정은 브라우저에 저장).
- 파일이 없으면 아무 소리도 나지 않으며 오류도 없습니다(무음).
